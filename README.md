# daft_punk

> Daft Punk discography chord and note explorer

## Controls

- **E1** — select album (HOMEWORK / DISCOVERY / HUMAN AFTER ALL / RANDOM ACCESS MEMORIES)
- **E2** — select song
- **E3** — arpeggiator speed (or octave when arp off)
- **K2** — toggle arpeggiator on/off
- **K3** — cycle arp style (UP / DN / U+D / RND / DRNK / BRST)

## Grid

Left half (cols 1–7): chord buttons. Right half (cols 9–16): note buttons. Chord expansion + arpeggiator absorbs all held notes (max 10). Direct play when arp is off.

## Requirements

- norns
- grid (16×8, required)
- MIDI device (optional)
- engine: MollyThePoly (internal synth)

## Install

```
;install https://github.com/jamminstein/daft_punk
```
