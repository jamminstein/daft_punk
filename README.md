# daft_punk

Daft Punk Anatomy — complete discography chord and note explorer with arpeggiator. Browse albums and songs, play chords on grid left half, notes on right half. 6 arp styles.

## Description

Daft Punk Anatomy brings Daft Punk's complete discography to norns. Explore albums and songs with their harmonic analyses. The grid splits into two regions: play chords on the left half, explore individual notes on the right half. Six distinct arpeggiator styles add rhythmic movement and variation to your playing.

A tribute instrument for fans and musicians inspired by electronic music's most iconic duo.

## Installation

`;install https://github.com/jamminstein/daft_punk`

## Controls

- **E1**: Select album
- **E2**: Select song
- **E3**: Adjust arpeggiator speed/octave
- **K2**: Toggle arpeggiator on/off
- **K3**: Cycle arpeggiator style

### Grid (16x8)
- **Left half (cols 1-8)**: Chord buttons
- **Right half (cols 9-16)**: Individual note buttons
- **Rows 1-8**: Playable note/chord range

## Arpeggiator Styles

1. **UP**: Ascending note sequence
2. **DN**: Descending note sequence
3. **U+D**: Up then down
4. **RND**: Random note sequence
5. **DRNK**: Drunk walk (random walk with inertia)
6. **BRST**: Burst (rapid repeating notes)

## Features

- Complete Daft Punk discography database
- Chord and note analysis per song
- 6 distinct arpeggiator modes
- Real-time grid performance
- Octave control
- MIDI output on channel 1

## Requirements

- norns
- Grid 16x8
- Built-in MollyThePoly engine
